# Introduction to Test and Behavior-Driven Development (Coursera)

This project is part of a learning exercise focused on **Test-Driven Development (TDD)** and **Behavior-Driven Development (BDD)** using Node.js.

## 🚀 Features

- REST API for managing products
- Unit testing using test frameworks
- Behavior-driven testing using feature files
- Organized project structure for scalability

## 📁 Project Structure


├── features/ # BDD feature files (Gherkin)
│ └── step_definitions/ # Step implementations
├── public/ # Frontend files (HTML, CSS, JS)
├── src/
│ ├── models/ # Data models
│ ├── routes/ # API routes
│ └── app.js # Main application logic
├── tests/ # Unit & integration tests
├── server.js # Entry point
├── package.json
└── README.md


## 🧪 Testing Approach

### ✅ Test-Driven Development (TDD)
- Writing tests before implementing features
- Ensuring code reliability and maintainability

### ✅ Behavior-Driven Development (BDD)
- Using `.feature` files to describe application behavior
- Writing human-readable test scenarios
